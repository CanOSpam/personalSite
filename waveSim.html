Wave Simulator

Wave Simulator was a practicum project for Lumerical Solutions that we worked on in our third term. The simulator itself was a desktop application that used maxwell's equations to simulate and show how a wave propagates through space and obstacles in 2D.

 

The project was built using Qt, a GUI framework for C++. We used Qt to visualize the data provided by a solver module built by Lumerical Solutions. We added a number of features to the simulator, including the ability to add rectangles and circles that reflect the wave, single time-step simulation for increased accuracy in visualization, and saving and loading simulator states so simulation can be paused and continued later.

 

Starting this project with little knowledge of both Qt and C++ was a challenge, but my team and I persevered and learned the necessary skills in the four months we had. We learned a great deal about the technologies we used in the process. We also learned a lot about project management in the long term. This project lasted 4 months, which was significantly longer ranging than other projects we had worked on for school previously. The long reach and (relatively) large scope of this project presented a number of unique issues that we overcame. We eventually made extensive use of an agile project board, which helped considerably with project management.
